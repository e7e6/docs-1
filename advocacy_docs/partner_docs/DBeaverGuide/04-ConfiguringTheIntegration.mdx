Implementing DBeaver with EDB Postgres Advanced Server/EDB Postgres Extended Server requires the following components: 

- A running EDB Postgres Advanced Server / EDB Postgres Extended Server instance
- DBeaver
- EDB JDBC Driver

The following steps show how to configure DBeaver for EDB Postgres Advanced and EDB Postgres Extended:

## EDB Postgres Advanced Server

1. Download EDB JDBC Driver (by default, the Driver should download into the directory `C:\Program Files\edb\jdbc`)
2. Setup DBeaver with the EDB JDBC Driver. This can be done using 2 methods:

    **Method 1:** Change Driver for the default `EnterpriseDB` type
        a. Select `EDB` as database type and click `Next`. 
        <p align="center">
        <img  src="Images/ConnectToADatabase.png">
        </p>

        b. Click `Edit Driver Settings`. Remove the jar file path that DBeaver already added and replace it with the correct `edb-jdbc18.jar` file from JDBC installation path (**Windows:** `C:\Program Files\edb\jdbc\edb-jdbc18.jar`, **Linux:** `/user/edb/jdbc/edb-jdbc18.jar`).
        <p align="center">
        <img  src="Images/EditDriverSettings.png">
        </p>

    **Method 2:** Add Custom Driver for EDB JDBC
      a. From the `Database` menu select `Driver Manager` and click `New`.This will launch the `Driver Manager` dialog: `Database -> Driver Manager`. 
      b. The `Create new driver` screen will appear. 
      c. Enter the values in the fields in the `Create new driver` screen as shown below:
        i. `Driver Name`: edb (Name it anything you want)
        ii. `Driver Type`: PostgreSQL
        iii. `Class Name`: com.edb.Driver
        iv. `URL Template`: jdbc:edb://{host}:{port}/{database}
        v. `Default Port`: 5444
      d. In the `Libraries` section click `Add File` and browse to select `edb-jdbc18.jar`
      e. For `Driver Class` click `Find Class`. This will show `com.edb.Driver` in the combo box. 
      f. Select `com.edb.Driver` and click `OK`. This will add the driver. 
      <p align="center">
      <img  src="Images/AddCustomDriver.png">
      </p>

3. From the `Window` menu select `Database Navigator`. This will open the `Database Navigator` window.
4. Click the icon for `New Database Connection`. The `Connect to database` dialog will appear.
5. Select `edb` or `EnterpriseDB` and click `Next`.
<p align="center">
<img  src="Images/SelectYourDatabase.png">
</p>
6. Provide connection information such as user, password, host, port and database. For example, enter `edb` as the database; the port should be according to where the selected server is running. The host can be the localhost.
7. If youâ€™re a Windows user, select the EDB Postgres Advanced Server version from the `Local Client` combination. This is not required for Linux.
<p align="center">
<img  src="Images/ConnectionSettings.png">
</p>
8. Click `Test Connection`. The connection should be successful.
<p align="center">
<img  src="Images/ConnectionTest.png">
</p>

## EDB Postgres Extended 
 
Select EDB `Postgresql` as database type and click `Next`.
On the  Connection Settings screen, click `Test Connection` to verify connectivity to database server.
