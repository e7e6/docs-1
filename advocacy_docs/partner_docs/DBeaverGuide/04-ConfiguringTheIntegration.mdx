Implementing DBeaver with EDB Postgres Advanced and EDB Postgres Extended servers requires the following components: 

- A running EDB Postgres Advanced Server / EDB Postgres Extended Server instance
- DBeaver

The following steps show how to configure DBeaver for EDB Postgres Advanced and EDB Postgres Extended servers:

## EDB Postgres Advanced Server

!!! Note
    Because DBeaver comes configured with the EDB JDBC connector, you donâ€™t need any additional configuration. 

1. Launch DBeaver on your computer. 
2. Go to the `Database` drop down menu in the top left, and select `New Database Connection`. In the `All` tab, select `EnterpriseDB` and click `Next`.    
<p align="center">
<img  src="Images/NewDatabaseConnection.png">
</p>

You can also connect to your EDB database instance by selecting `New Connection from JDBC URL` from the `Database` menu. DBeaver will automatically select EnterpriseDB in the `Drivers` field when the EDB JDBC URL is provided in the `JDBC URL` field. Click `Proceed`.
<p align="center">
<img  src="Images/JDBCURL.png">
</p>

3. Enter your username and password. 
<p align="center">
<img  src="Images/EDBConnectionSettings.png">
</p>

4. Click `Test Connection`. The connection should be successful. 
<p align="center">
<img  src="Images/ConnectionSuccessful.png">
</p>

5. Click `Ok` on this window and then click `Finish` on the `Connection Settings` window. The connection is then created and shown in the left-side tree. 
<p align="center">
<img  src="Images/ConnectionOnLeftSide.png">
</p>
