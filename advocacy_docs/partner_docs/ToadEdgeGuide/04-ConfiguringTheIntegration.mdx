Implementing Toad Edge with EDB Postgres Advanced Server/EDB Postgres Extended Server requires the following components: 

- A running EDB Postgres Advanced (EPAS) / EDB Postgres Extended instance
- Toad Edge
- EDB JDBC Driver

The following steps show how to configure Toad Edge for EDB Postgres Advanced and EDB Postgres Extended servers:

**EDB Postgres Advanced Server:**

1. Launch the Toad Edge application
2. On the main menu, go the `Connect` dropdown and select `New Connection`
3. In the `New Connection` screen, enter the following values: 
        - `Hostname`
        - `Port`
        - `Database`
        - `Username`
        - `Password`
4. Select `EDB Postgres Advanced Server` on the left pane. 
5. Click `Test Connection` to verify connectivity to the database server.
6. If the connection is successful, click `Connect`.

**EDB Postgres Extended:**

1. Launch the Toad Edge application
2. On the main menu, go the `Connect` dropdown and select `New Connection`
3. In the `New Connection` screen, enter the following values: 
        - `Hostname`
        - `Port`
        - `Database`
        - `Username`
        - `Password`
4. Select Postgres on the left pane. 
5. Click Test Connection to verify connectivity to the database server.
6. If the connection is successful, click Connect.
