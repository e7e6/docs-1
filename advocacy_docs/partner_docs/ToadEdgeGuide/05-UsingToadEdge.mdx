## Known Issues: 

**Password should be at least 14 characters:**

Check Issue 14740 **Known Issues** section in (Toad Edge 2.4 - Release Notes)[https://support.quest.com/technical-documents/toad-edge/2.4/release-notes/2#TOPIC-1705496]

The following error message is displayed when connecting to any EDB Postgres Advanced Server version when password is less than 14 characters: 

*Password should be at least 14 characters in FIPS mode.*

**Workaround:** Close ToadEdge and go to `C:\Program Files\Quest Software\Toad Edge\ToadEdge.ini` file and change the value of -`Dorg.bouncycastle.fips.approved_only` to `false`.

`1-Dorg.bouncycastle.fips.approved_only=false`


**Postgres and EDB Postgres Advanced Server v14:**

Officially Toad Edge version 2.4.1 supports EDB Postgres Advanced Server <= 13

Check EnterpriseDB in **Database Requirements** section in (Toad Edge 2.4 - Release Notes)[https://support.quest.com/technical-documents/toad-edge/2.4/release-notes/2#TOPIC-1705496]
